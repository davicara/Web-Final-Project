About Us

Built as part of my Certificate IV in IT (Programming) Mobile App Development unit, this music application was developed using .NET MAUI, bringing together modern cross-platform development with the power of the MusicBrainz API. The app lets you search for your favourite artists, songs, and albums, with a clean and intuitive interface that lets you click through to detailed pages for each result. Built entirely in C# and XAML, it demonstrates real-world mobile development skills including API integration, navigation, and responsive UI design across multiple pages.

MusicBrainz API

MusicBrainz is a free and open-source music encyclopedia that provides one of the most comprehensive music databases in the world. Its API gives developers access to an enormous collection of metadata covering artists, recordings, releases, and albums — all contributed and maintained by a global community of music lovers. Unlike many music APIs, MusicBrainz is completely free to use with no API key required, making it a fantastic resource for developers. Every entry in the database is identified by a unique ID, ensuring accurate and consistent data no matter what you're searching for.
